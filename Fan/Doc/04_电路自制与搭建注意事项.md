# 第四阶段：电路自制与洞洞板搭建注意事项

## 一、整体布局原则

洞洞板分区焊接，从左到右依次：

```
[12V 动力区] | [降压电源区] | [STM32 主控区] | [传感器接口区]
```

各区之间尽量留 1~2 排孔隔离，避免强电干扰弱电。

---

## 二、12V 动力线处理

### 2.1 线材选择
- 12V 风扇电流最大约 0.5~1A，使用 **≥22AWG（约 0.35mm²）** 导线
- 信号线（传感器、I2C、GPIO）使用 26~28AWG 细线

### 2.2 LR7843 MOS 管驱动注意事项

```
                 12V (动力)
                  │
                [风扇 12V]
                  │
                  D (漏极)
                ┌────┐
           G ───│LR78│
                │ 43 │
                └────┘
                  S (源极)
                  │
                 GND
```

- **G 极（栅极）**：接 STM32 PWM 引脚（PA8），中间串联 **100Ω 电阻**，防止振荡
- **G-S 之间**：焊接 **10kΩ 下拉电阻** 到 GND，防止 MCU 复位时 G 极悬空导致风扇误启动
- **续流二极管（强制要求）**：在风扇两端并联 **1N4007 或 FR107 续流二极管**（负极接 12V，正极接 MOS 漏极），吸收反电动势脉冲，保护 MOS 管

### 2.3 反电动势保护

风扇是感性负载，关断瞬间会产生反向高压尖峰。不加续流二极管可能损坏 MOS 管甚至 STM32：

```c
  风扇 (12V)
   (+)──────┬──────(D)
            │       |
           [D]   LR7843   ← D 为 1N4007，方向：负极朝上
            │       |
   (-)──────┴──────(S)──── GND
```

## 三、电源分配方案

### 3.1 LM2596S 降压模块
- 输入：12V
- 输出：5V，调节电位器直到万用表显示 5.0V
- **在输出端并联 100μF 电解电容 + 0.1μF 陶瓷电容**，改善滤波

### 3.2 3.3V 供电
- STM32 最小系统板板载 AMS1117-3.3，从板上的 3.3V 引脚直接取电给传感器
- AHT20、MLX90614、OLED：均使用 3.3V 供电
- ACS712 模块、MQ-2 模块：使用 5V 供电（注意！模拟输出电压最高可达 5V，需**分压后再接 STM32 ADC**）

### 3.3 MQ-2 / ACS712 输出分压电路

模块输出最高 5V，STM32 ADC 只能承受 3.3V，必须分压：

```
模块输出 ──[2kΩ]──┬──→ PA0/PA1 (ADC)
                  │
                [3kΩ]
                  │
                 GND
```

分压比 = 3/(2+3) = 0.6，即 5V 降到 3V，满足 3.3V ADC 输入限制。

## 四、I2C 总线处理

### 4.1 上拉电阻
- I2C 总线（SDA/SCL）必须有上拉电阻
- 如果挂 3 个设备（AHT20 + MLX90614 + OLED），推荐使用 **2.2kΩ** 上拉到 3.3V
- 通常 OLED 模块板上已有 4.7kΩ 上拉，AHT20 模块也有，多个并联后等效值降低，**总阻值应在 1kΩ~5kΩ 之间**

### 4.2 地址冲突检查
- AHT20 地址：0x38（固定）
- MLX90614 地址：0x5A（默认，可改）
- OLED SSD1306 地址：0x3C 或 0x3D（取决于 SA0 引脚）
- 三者地址不冲突，可挂在同一 I2C 总线

---

## 五、抗干扰措施

### 5.1 电源滤波
每个模块的 VCC 引脚就近焊接 **0.1μF 陶瓷电容** 到 GND，去除高频噪声。

### 5.2 地线处理
- 12V GND 和 3.3V/5V GND 在**电源输入端单点共地**
- STM32 AGND（模拟地）和 DGND（数字地）在 MCU 就近单点连接

### 5.3 ADC 引脚保护
- MQ-2 和 ACS712 的模拟信号线靠近 ADC 引脚附近焊接一个 **100pF 陶瓷电容** 到 GND，滤除高频干扰
- 信号线尽量短，避免与 12V 动力线平行走线

### 5.4 MOS 管栅极保护
- G 极串联 **100Ω 电阻**（防振荡）
- G-S 间 **10kΩ 下拉**（防悬空）
- 若环境干扰严重，可在 G-S 间再加 **100pF** 电容

---

## 六、洞洞板焊接步骤建议

1. **先焊电源部分**：放置 DC 座、LM2596S、滤波电容，上电测量 5V/3.3V 正常后再继续
2. **焊接 STM32 最小系统板座**：用排针座，方便拔插
3. **焊接 MOS 管驱动电路**：LR7843、栅极电阻、续流二极管
4. **焊接传感器接口排针**：为每个模块留 4P/5P 插针座，模块用杜邦线连接（便于调试更换）
5. **焊接分压电路**：MQ-2 和 ACS712 输出分压电阻
6. **焊接 I2C 上拉电阻**：2.2kΩ × 2
7. **焊接蜂鸣器、LED、按键**：加限流电阻（LED：100Ω，蜂鸣器直接驱动或加 S8550 三极管）

## 七、上电前安全检查清单

- [ ] 确认 LM2596S 输出电压已调节到 5.0V
- [ ] 确认所有模块 VCC/GND 方向正确
- [ ] MOS 管 G 极下拉电阻已焊
- [ ] 续流二极管方向正确（负极接 12V 侧）
- [ ] 分压电阻值正确
- [ ] I2C 上拉电阻已接
