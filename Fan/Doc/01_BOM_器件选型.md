## 二、关键器件选型理由

### 2.1 MLX90614ESF-BAA — 移动热水检测核心

**为什么选它：**
- **宽视场角 (FOV = 90°)**：单个传感器可覆盖大面积区域，无需多传感器阵列
- **I2C 接口**：仅占用 2 个引脚，与 OLED、AHT20 共用总线
- **自带环境温度补偿**：芯片内部有环境温度寄存器 `Ta`，可直接用作室温参考
- **精度 ±0.5℃**：满足 5℃ 温差判断需求
- **3.3V 直接供电**：无需电平转换

**移动花洒检测原理：**
```
┌─────────────────────────────────┐
│         MLX90614 (FOV=90°)      │
│         安装在天花板/墙壁上       │
│              ↓                  │
│    ┌─────────────────────┐      │
│    │   90° 视场锥形区域    │      │
│    │                     │      │
│    │   花洒在此区域内移动   │      │
│    │   → 只要有热水辐射    │      │
│    │   → T_obj > T_a + 5  │      │
│    │   → 判定"热水开启"    │      │
│    │                     │      │
│    │   花洒移出 或 关水     │      │
│    │   → T_obj ≤ T_a + 5  │      │
│    │   → 判定"热水停止"    │      │
│    └─────────────────────┘      │
���─────────────────────────────────┘
```

**关键机制：**
1. MLX90614 测量的是视场内**所有物体的平均红外辐射**
2. 90° 宽视场意味着花洒在较大范围内移动时，只要热水区域占视场的一定比例，`T_obj` 就会显著升高
3. 软件端做**滑动平均滤波** (取连续 5 次采样的平均值)，消除花洒快速移动时的抖动
4. 设置 **迟滞阈值**：开启判断 `T_obj >= T_a + 5`，关闭判断 `T_obj < T_a + 3`，防止边界反复触发

### 2.2 ACS712-5A — 堵转检测（替代 INA219）

**选择理由：**
- **比 INA219 便宜**：ACS712 模块约 5 元，INA219 约 12 元
- **纯模拟输出**：AO 接 STM32 ADC 即可，无需 I2C（减少总线负载）
- **原理简单**：输出电压与电流线性关系 → `V_out = 2.5V + 0.185V/A × I`
- **5A 量程**：12V 风扇典型工作电流 0.1~0.5A，堵转电流 1~3A，5A 量程恰好覆盖

**堵转判断逻辑：**
- 正常运行电流：0.1 ~ 0.5A
- 堵转特征：电流突增到 > 1.5A 且持续 > 500ms → 判定堵转 → 立即停机保护

### 2.3 MQ-2 — 有毒气体检测

- 对**丁烷 (打火机气体)** 灵敏度极高
- 预热时间约 20 秒（上电后软件做预热等待）
- AO 模拟输出接 ADC：空气中约 0.1~0.4V，检测到丁烷时可达 2.0V 以上
- 设定 ADC 阈值：超过约 1.0V (ADC 值约 1241) 即触发气体报警

### 2.4 AHT20 vs DHT11

| 对比项 | AHT20 | DHT11 |
|--------|-------|-------|
| 接口 | I2C | 单总线 |
| 温度精度 | ±0.3℃ | ±2℃ |
| 湿度精度 | ±2% | ±5% |
| 价格 | 4元 | 3元 |
| 推荐度 | ★★★★★ | ★★★ |

**结论**：AHT20 精度远高于 DHT11，且 I2C 接口更方便复用总线，仅贵 1 元，强烈推荐。

### 2.5 LR7843 MOS 驱动模块

- **逻辑电平兼容**：3.3V PWM 信号可直接驱动（MOSFET 栅极阈值约 2.5V）
- **大电流能力**：持续 10A，远超风扇需求
- **价格极低**：约 3 元
- **接线简单**：VIN/GND/SIG 三线制

## 三、器件采购建议

1. **统一在淘宝/拼多多搜索"模块"关键词**，购买已焊好的成品模块，节省焊接时间
2. MLX90614 务必选 **BAA 后缀版本**（90° 视场角），不要买 DCI 版本（5° 窄视场）
3. ACS712 选 **5A 版本**，不要选 20A/30A 版本（量程太大、分辨率不够）
4. OLED 选 **I2C 4 针版本**（VCC/GND/SCL/SDA），不要买 SPI 7 针版本
5. 建议多买 1~2 个备用的最小系统板，以防烧毁
