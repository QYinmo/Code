# 硬件连接、电源分配与引脚映射

> 项目：简易智能排气系统 | 主控：STM32F103C8T6

## 一、电源分配方案

```
                    ┌──────────────┐
  12V 2A 适配器 ───→│  DC 电源座    │
                    └──────┬───────┘
                           │ 12V
                           ├─────────────────→ 排气风扇 VCC (4线风扇 12V 直连, 无需 MOS)
                           │
                           ├─────────────────→ ACS712 电流检测模块 VCC (串联在风扇供电回路)
                           │
                    ┌──────┴───────┐
                    │  LM2596S     │
                    │  12V → 5V    │
                    └──────┬───────┘
                           │ 5V
                           ├─────────────────→ MQ-2 气体传感器 VCC
                           ├─────────────────→ 有源蜂鸣器 VCC
                           ├─────────────────→ ACS712 模块 VCC (逻辑供电)
                           │
                    ┌──────┴───────┐
                    │  STM32 板载   │
                    │  AMS1117     │
                    │  5V → 3.3V   │
                    └──────┬───────┘
                           │ 3.3V
                           ├─────────────────→ STM32F103C8T6
                           ├─────────────────→ MLX90614 (3.3V)
                           ├─────────────────→ AHT20 #1 室内 (3.3V)
                           ├─────────────────→ AHT20 #2 室外 (3.3V)
                           ├─────────────────→ OLED SSD1306 (3.3V)
                           ├─────────────────→ LED 指示灯 (经限流电阻)
                           └─────────────────→ 按键上拉
```

**关键注意事项：**
- STM32 最小系统板的 5V 引脚直接接 LM2596S 输出，板载 AMS1117 再降到 3.3V
- MQ-2 必须接 5V（加热电阻需要 5V）
- LR7843 模块**已移除** (4线风扇方案), PA8 PWM 信号直接接风扇蓝线 (PWM 输入)
- 风扇 12V 红线直连电源, 黑线接 GND, 黄线(TACH)接 PA7, 蓝线(PWM)接 PA8
- **每个模块的 GND 必须与 STM32 GND 共地！**

## 四、MQ-2 / ACS712 分压电路说明

MQ-2 和 ACS712 模块的 AO 输出范围为 0~5V，而 STM32 ADC 最大输入为 3.3V，**必须加分压电路**：

```
MQ-2 AO ──── R1 (2KΩ) ────┬──── PA0 (ADC)
                           │
                       R2 (3KΩ)
                           │
                          GND

ACS712 AO ── R3 (2KΩ) ────┬──── PA1 (ADC)
                           │
                       R4 (3KΩ)
                           │
                          GND

分压比: V_adc = V_ao × 3K / (2K + 3K) = V_ao × 0.6
当 V_ao = 5V 时, V_adc = 3.0V (安全范围内)
```

> ACS712-5A 在 5A 满量程时输出 = 2.5 + 0.185×5 = 3.425V, 分压后 = 3.425×0.6 = 2.055V, 安全。

## 五、风扇驱动 + 测速电路详图

```
        12V ─────────────────────────┐
                                  │
                             ┌────┴────┐
                             │  4线风扇  │
                             │  12V DC  │
                             │  红=12V   │──── 直接接 12V (不经过 MOS!)
                             │  黑=GND   │──── 公共 GND
                             │  黄=TACH  │──→ PA7 (TIM3_CH2 输入捕获)
                             │  蓝=PWM   │──← PA8 (TIM1_CH1 PWM 25kHz, 可串联 1KΩ)
                             └────┬────┘
                                  │
                            ┌─────┴─────┐
                            │  ACS712    │
                            │  (串联)    │
                            │  IP+ IP-  │
                            └─────┬─────┘
                                  │
                                 GND
```

> **注意**: 4线风扇方案已移除 LR7843 MOS 模块和续流二极管。PA8 的 PWM 信号直接接到风扇蓝线。
> 如风扇 PWM 输入有 5V 上拉, PA8 需串联 1KΩ 保护电阻, 防止 5V 倒灌到 STM32 GPIO。

**风扇 TACH 信号说明：**
- 4线风扇的黄色线为 TACH 输出, 开漏/开集电极结构 (与3线完全一致)
- 需要上拉电阻 (一般风扇模块内置, 若无则外接 10KΩ 上拉到 3.3V)
- 每转输出 2 个下降沿脉冲
- 接 PA7 (TIM3_CH2), 通过输入捕获测量脉冲周期计算 RPM

## 六、接线核查清单

焊接/接线完成后，逐项检查：

- [ ] 12V 电源正负极确认无误
- [ ] LM2596S 输出电压调节到 5.0V (用万用表确认)
- [ ] STM32 的 5V 引脚接到 LM2596S 输出
- [ ] 所有模块 GND 汇总到公共地
- [ ] I2C1 总线 PB6/PB7 外接 4.7KΩ 上拉电阻到 3.3V
- [ ] I2C2 总线 PB10/PB11 外接 4.7KΩ 上拉电阻到 3.3V
- [ ] MQ-2 分压电路已焊好 (2K + 3K)
- [ ] ACS712 分压电路已焊好 (2K + 3K)
- [ ] ~~1N4007 续流二极管~~ (4线风扇方案不需要)
- [ ] 按键一端接 GPIO，另一端接 GND
- [ ] LED 限流电阻 (1KΩ) 已串联
- [ ] 蜂鸣器信号端接 PB5
- [ ] 风扇 TACH 线接 PA7 (如无上拉需外接 10KΩ)
- [ ] 室外 AHT20 的 SCL/SDA 接 PB10/PB11 (非 PB6/PB7)
