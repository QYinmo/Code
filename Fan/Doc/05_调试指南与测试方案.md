# 调试指南与测试方案

## 一、调试环境准备

| 工具 | 用途 |
|------|------|
| ST-Link V2 / J-Link | 程序下载和调试 |
| USB-TTL（CH340） | 串口 Debug_Print 输出 (PA9/PA10, 115200) |
| 万用表 | 电压/电阻测量 |
| 热水（约 50℃）+ 冷水 | 模拟热水源 |
| 打火机 | 模拟可燃气体（不点火，只充气） |
| 细棍/手指 | 插入风扇模拟堵转 |

## 二、分阶段上电调试步骤

### 步骤 1：纯电源测试（不接 MCU）
1. 接入 12V 适配器
2. 万用表测 LM2596S 输出端 → 应为 **5.0V ± 0.1V**
3. 测 STM32 最小系统板 3.3V 引脚 → 应为 **3.3V ± 0.1V**
4. 检查无烫手元件、无焦糊味 → 进入下一步

### 步骤 2：烧录程序 + 串口验证
1. Keil 编译工程 (F7), 通过 ST-Link 下载 (F8)
2. 连接 USB-TTL 到 PA9(TX)/PA10(RX)，波特率 115200
3. 复位 MCU 后串口应输出：
   ```
   ACS712 零点: 1862
   Flash: 参数加载成功     (或 "首次启动, 使用默认参数")
   系统启动... MQ-2 非阻塞预热中
   MQ-2 就绪
   State: 1 -> 0           (PREHEAT -> IDLE)
   ```
4. OLED 显示启动画面 "Smart Exhaust V3"，预热倒计时后显示主界面

### 步骤 3：传感器验证

**室内 AHT20 (I2C1):**
- OLED 第 1 行显示 `In:25.3C 58%`
- 若末尾显示 `!` 表示传感器离线

**室外 AHT20 (I2C2):**
- OLED 第 2 行显示 `Out:22.1C 45%`
- 离线时显示 `Out:OFFLINE`
- 连续失败后串口输出 `I2C2 总线恢复` (自动恢复)

**MLX90614 (I2C1):**
- OLED 第 3 行显示水温 `Tw:26.0`
- 用手捂住镜头 → Tw 应升到 33~36℃
- 末尾 `!` 表示 Tobj 离线

**风扇测速 (PA7):**
- OLED 第 4 行显示 `RPM:xxxx`
- 风扇转动时应显示 500~3000 范围内的值
- 风扇停转超过 2 秒后显示 `RPM:0`

### 步骤 4：热水检测联动测试

**测试方法：用热水杯模拟热水源**

1. 准备一杯约 50℃ 热水，放在 MLX90614 视野内（正对镜头 30~50cm）
2. 串口输出：
   ```
   State: 0 -> 3    (IDLE -> HOT_WATER)
   ```
3. OLED 显示 `MODE:HOT WATER`，PWM 值根据 PID 计算输出
4. 风扇开始转动，OLED 第 4 行显示当前占空比和转速
5. OLED 第 3 行 `dAH:x.x` 显示室内外绝对湿度差

**PID 调速验证：**
- 室内外绝对湿度差越大 → PWM 越高
- 室外传感器离线时 → 固定 40% 占空比 (显示 `MODE:HOT W(DEG)`)
- 室温 < 18℃ → 限制最大 50%
- 最低保障 30% (防止 PID 输出为 0 但蒸汽未到传感器)

**延时除湿测试：**
1. 移走热水杯 → 串口输出 `State: 3 -> 4` (HOT_WATER → DELAY_STOP)
2. OLED 第 6 行显示 `B:30s F:90s` (基础/强制倒计时)
3. 基础延时结束后检查绝对湿度收敛:
   - `H:OK` = 已收敛 → 停机
   - `H:HI` = 未收敛 → 继续运行直到强制超时
4. 停机后 → `State: 4 -> 0` (DELAY_STOP → IDLE)

### 步骤 5：气体报警测试（最高优先级）

**测试方法：使用打火机充气（不点火！）**

1. MQ2 预热完成后 (上电等 20s)
2. 将打火机靠近 MQ-2 传感器，按下充气阀释放少量气体（1~2 秒）
3. 串口应输出 `State: x -> 5` (→ GAS_ALARM)
4. OLED 显示 `MODE:GAS ALARM`，`PWM:100%`
5. 蜂鸣器 200ms 间隔急促鸣叫，红色 LED 闪烁
6. 风扇全速运行
7. **验证最高优先级**：即使此时无热水，风扇仍全速运行
8. 移走打火机，等待气体散去 (ADC < 1800 连续 5 次) → 报警自动解除

> **安全提醒**：室内测试时保持通风，不得在密闭空间大量释放丁烷气体！

### 步骤 6：堵转故障检测测试

1. 先手动令风扇以 50% 以上转速运行（触发热水检测）
2. 串口应输出 ACS712 峰值和堵转阈值：
   ```
   ACS712 峰值:85 堵转阈值:212
   ```
3. 用细棍轻轻阻挡风扇叶片 3~5 秒（模拟堵转）
4. 串口输出 `State: x -> 6` (→ STALL_FAULT)
5. OLED 显示 `MODE:STALL ERR`
6. 红灯 + 蜂鸣器 500ms 间隔慢报警
7. 移除阻挡，按 KEY1 复位故障

**堵转判据验证：**
- 判据 1: 电流超阈值持续 > 500ms → 触发
- 判据 2: RPM < 100 持续 > 3s → 触发
- PWM ≤ 30% 时不检测 (避免低速误判)

### 步骤 7：按键功能测试

| 按键 | 引脚 | 功能 | 验证方法 |
|------|------|------|----------|
| KEY1 (PA4) | 故障复位 | 在 STALL_FAULT 状态下按下, 状态回到 IDLE |
| KEY2 (PA5) | 延时 +10s | OLED 第 6 行延时时间增加, 长按连发 |
| KEY3 (PA6) | 延时 -10s | OLED 第 6 行延时时间减少, 长按连发 |

延时调节范围: 10~120 秒, 步进 10 秒。修改后 3 秒内无新操作则自动保存到 Flash, 串口输出 `延时已保存: xx s`。

### 步骤 8：Flash 参数持久化验证

1. 按 KEY2/KEY3 修改延时时间
2. 等待 3 秒, 串口输出保存成功
3. 按复位键重启 MCU
4. 串口输出 `Flash: 参数加载成功`
5. OLED 显示的延时时间应为上次保存的值

## 三、OLED 显示界面说明

**正常运行界面 (7 行)：**
```
MODE:IDLE               ← 当前状态
In:25.3C 58%            ← 室内温湿度 (! = AHT20离线)
Out:22.1C 45%           ← 室外温湿度 (或 Out:OFFLINE)
dAH: 3.2 Tw:45.1       ← 绝对湿度差 + 水温 (! = MLX离线)
PWM: 52% RPM:1200       ← 占空比 + 实时转速
Gas:1234 I:0.35A        ← MQ-2 ADC + 风扇电流
Dly:30s     Gas:NOR     ← 延时设定 + 气体状态
```

**延时除湿阶段第 6 行变为：**
```
B:25s F:65s      H:HI   ← 基础/强制倒计时 + 湿度收敛状态
```
- `B:xxs` = 基础延时剩余 (结束后显示 `B:OK`)
- `F:xxs` = 强制超时剩余
- `H:OK` = 绝对湿度已收敛 | `H:HI` = 仍高于室外

**各模式下的状态显示：**

| 状态 | MODE 显示 | LED | 蜂鸣器 |
|------|----------|-----|--------|
| PREHEAT | `MODE:PREHEAT xxs` | 黄灯慢闪 | 静音 |
| IDLE | `MODE:IDLE` | 绿灯常亮 | 静音 |
| HOT_WATER | `MODE:HOT WATER` 或 `MODE:HOT W(DEG)` | 黄灯常亮 | 静音 |
| DELAY_STOP | `MODE:DEHUMID` | 绿+黄灯常亮 | 静音 |
| GAS_ALARM | `MODE:GAS ALARM` | 红灯闪烁 | 200ms急促 |
| STALL_FAULT | `MODE:STALL ERR` | 红灯常亮 | 500ms慢鸣 |

## 四、赛场演示流程建议

1. **通电初始化**（20s）：MQ-2 预热，OLED 显示启动画面和倒计时
2. **热水演示**：放入热水杯 → 风扇启动，介绍 PID 绝对湿度闭环控制
3. **移动花洒演示**：移动热水杯模拟花洒移位 → 90° 宽视场仍能检测
4. **撤走热水**：讲解延时除湿 + 绝对湿度收敛判断
5. **调节延时**：按 KEY2/KEY3 现场调节，OLED 实时显示，Flash 自动保存
6. **气体报警演示**（高潮）：打火机充气 → 立即全速报警
7. **复位演示**：气体散去后自动恢复
8. **堵转演示**：阻挡风扇 → 自动保护停机 → KEY1 复位

## 五、常见问题排查

| 现象 | 可能原因 | 解决方法 |
|------|----------|----------|
| I2C1 设备初始化失败 | SDA/SCL 接错或无上拉 | 检查 PB6/PB7 接线和 4.7kΩ 上拉电阻 |
| 室外 AHT20 始终 OFFLINE | I2C2 未初始化或接线错误 | 检查 PB10/PB11 接线 + CubeMX 是否配置了 I2C2 |
| MLX90614 读数为 0 | PEC 校验失败 | 检查 I2C 时序，降低速度到 100kHz |
| MQ-2 始终报警 | 未充分预热 | 等待 20s 预热或调高 GAS_ALARM_THRESHOLD |
| 风扇不转 | MOS 管 G 极无驱动 | 万用表测 PA8 PWM 波形，检查 G 极电阻 |
| RPM 始终为 0 | TIM3 中断未使能 | 检查 CubeMX 是否开启 TIM3 全局中断 |
| RPM 始终为 0 | TACH 线未接或无上拉 | 检查 PA7 接线, 外接 10KΩ 上拉 |
| 堵转误触发 | ACS712 零点偏移 | 确保上电时风扇静止, 检查校准输出 |
| OLED 花屏 | I2C 地址错误 | 确认 SA0 引脚电平，尝试 0x3C 和 0x3D |
| Flash 保存失败 | Flash 页被锁定 | 检查 HAL_FLASH_Unlock 返回值 |
| 延时参数未保存 | 修改后不到 3s 就断电 | 等待串口输出 "延时已保存" 后再断电 |
| 系统复位/看门狗重启 | 主循环阻塞超过 4 秒 | 检查 I2C 通信是否超时, Debug_Print 输出是否阻塞 |
